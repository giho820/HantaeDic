부록1. 발음의 규칙 정리

  • 태국어의 성조(톤)는 주로 성조 부호(톤 마크)가 어느 글자에 붙었는가에 따라 그 억양이 다르게 발음됩니다.
  • 평성(-) : 평상시의 보통 높이의 억양으로 발음
  • 1성(＼,LOW) : 보통 소리에서 낮아지게 발음
  • 2성(∧,FALLING) : 올라가다가 낮아지게 발음
  • 3성(／,HIGH) : 보통 소리에서 높아지게 발음
  • 4성(∨,RISING) : 낮아지다가 올라가도록 발음

기타 발음시 주의사항

CONSONANT CLUSTER

1. C-1 : ท, ม, บ (ธ,ว,ห,ก,จ) →-อ
C-2 : ร → -ะ (*บริ)
예) ทรมาน → ทอ-ระ-มาน

C-1과 C-2가 결합하면 C-1에는 อ가 추가되고 C-2(ร)에는 단모음 아(ะ)가 붙는다(예외- บริ(버리))

2. C-1 : HC , MC → -ะ
C-2 : LC-D → ห-นำ
예) สนุก → สะ-หนุก ,ตลาด → ตะ-หลาด

(설명) C-1이 HC , MC이고 C-2가 악썬- 디여우면 C-1에는 단모음 아(ะ)가 추가, C-2 는 ห-นำ이 된다

3. C-1 : HC,MC,LC→ -ะ
C-2 : HC,MC,LC→보통
예) สบาย → สะ-บาย, เจริญ → จะ-เริญ

(설명) c1 이 HC , MC이고 c2 가 악썬- 디여우면 c1 에는 단모음 아(ะ)가 추가, c2 는 ห-นำ이 된다
DOUBLE FUNCTION

ผลไม้ → ผน-ละ-ไม

ล이 양쪽에 모두 영향을 주어 폴라마-이

** ร :

1.ทร → ซ : ทราบ

ทร는 써(ซ) 발음 (예→쌉-)

2.ศร → ศ : เศร้า

ศร는 써(ศ) 발음 (예→싸-오)

3.สร → ส : สร้าง

สร는 써(ส) 발음 (예→쌍-)

4.จร → จ : จริง (1개뿐)

จร는 쩌(จ) 발음 (예→찡)
기타

** รร

รร는 3가지발음이 된다

1. -ะ → ธรรม

단모음 아(ะ) (예→탐)

2. -ัน → กรรไกร

안(-ัน) (예→깐끄라이)

3. -ันระ → ภรรยา, สรรเสริญ

안라(-ันระ) (예→판라야-)

** C-ร : -อน
จราจร → จราจอน

다른 글자와 ร이 결합하면 언(อน)으로 발음(예→짜라-쩐)
